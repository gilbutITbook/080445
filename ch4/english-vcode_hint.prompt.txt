### 지시: 
영어 실력을 향상시킬 수 있는 아이디어를 생각해 줘.
단, 중간 추론 단계로서 아래 의사 코드의 동작을 모방해서 아이디어를 출력해.

### 힌트로 제공하는 의사코드: 
``` 
목표 = '영어 실력을 향상시킬 수 있는 아이디어 생각하기' 
while True: 
   # 아이디어를 Zero-shot으로 고안 
   아이디어1 = 아이디어 고안(목표, 관점) 
   # 영어를 잘하는 사람의 관점을 추가하여 다듬기 
   관점 = 영어를 잘하는 사람의 관점을 고안
   아이디어2 = 아이디어 다듬기(아이디어, 관점) 
   점수 = 10단계로 재미있는 아이디어인지 평가(아이디어2) 
   print(f'{관점} > {아이디어2} ({점수}점)') 
   if 점수 == 10: break 
```
